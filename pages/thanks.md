# Thanks

pota is a research and personal project built on top of other people's work. A few of the shoulders it stands on.

## Solid

The API and base components are heavily inspired by [Solid](https://www.solidjs.com/). pota differs to an extent based on personal preference, but the lineage is direct.

## oby & flimsy

The reactive core used to be pluggable, and these libraries — among other attempts — were explored as backends. The pluggable idea was eventually dropped, but the experiments shaped what pota ships today: [oby](https://github.com/vobyjs/oby) · [flimsy](https://github.com/fabiospampinato/flimsy).
