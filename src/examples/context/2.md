---
title: Functional context override
description: Run a callback with a temporarily overridden context value.
category: reactivity
api: context
import: pota
tags: [context, override]
---

Calling the context function as `Theme(newValue, fn)` runs `fn` with
the override applied; outside of `fn`, the previous value is
restored. Useful when you need the context override outside the JSX
tree (computing a derived value, in an `effect`, etc.).

```js
import { context } from 'pota'

const Theme = context('light')

console.log(Theme())  // 'light'

const result = Theme('dark', () => {
	return Theme('contrast', () => Theme())
})

console.log(result)   // 'contrast'
console.log(Theme())  // 'light' — restored
```
