---
title: Set an attribute reactively on a ref'd node
description: Apply an attribute imperatively on a captured DOM element.
category: props
api: setAttribute
import: pota
tags: [setAttribute, dom, imperative]
---

`setAttribute(node, name, value)` writes an attribute, unwrapping
reactive accessors automatically. `value === false`/`null`/`undefined`
removes the attribute; `true` sets it to the empty string. Reach for
this when the JSX prop form isn't an option — typically inside a
custom prop plugin or when working with a node from `ref()`.

```jsx
import { ref, render, setAttribute, signal } from 'pota'

function App() {
	const button = ref()
	const busy = signal(false)

	return (
		<div>
			<button
				use:ref={button}
				on:click={() => busy.update(b => !b)}
			>
				toggle
			</button>
			{() => {
				setAttribute(
					button(),
					'aria-busy',
					busy.read() ? 'true' : null,
				)
			}}
		</div>
	)
}

render(<App />)
```
