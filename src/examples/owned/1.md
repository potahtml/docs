---
title: Owner-bound async callback
description: Wrap a deferred callback so it doesn't run after disposal.
category: reactivity
api: owned
import: pota
tags: [owned, lifecycle, cleanup]
---

`owned(fn)` returns a function that is bound to the current owner.
When the owner is disposed (e.g. a parent component unmounts), the
returned wrapper becomes a no-op — useful for deferred work like
`setTimeout`, promise continuations, or websocket callbacks where
the work no longer matters once the scope is gone.

```jsx
import { owned, render, signal } from 'pota'

function App() {
	const status = signal('idle')

	function startSlowWork() {
		status.write('working…')
		const finish = owned(() => status.write('done'))
		setTimeout(finish, 2000)
	}

	return (
		<div>
			<p>status: {status.read}</p>
			<button on:click={startSlowWork}>start</button>
		</div>
	)
}

render(<App />)
```
