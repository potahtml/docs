---
title: Read the installed pota version
description: Display the package version at runtime.
category: utilities
api: version
import: pota
tags: [version, meta]
---

`version` is a string export — the version of the running `pota`
package. Useful for footer credits, debug overlays, and version-gated
feature checks.

```jsx
import { render, version } from 'pota'

function App() {
	return (
		<footer>
			powered by pota <code>v{version}</code>
		</footer>
	)
}

render(<App />)
```
