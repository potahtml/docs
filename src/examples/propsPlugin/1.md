---
title: Register a custom `use:` prop
description: Add a project-specific directive that any element can opt into.
category: props
api: propsPlugin
import: pota
tags: [propsPlugin, plugin, custom]
---

`propsPlugin(name, fn)` registers a handler invoked for every element
that declares `name` as a prop. Use it for behaviors that aren't
specific to a component — long-press, intersection observers,
draggable handles. By default the handler runs on the `onProps`
microtask (so children and refs are already in place).

```jsx
import { propsPlugin, render } from 'pota'

propsPlugin('use:longpress', (node, value) => {
	let timer
	const start = () => {
		timer = setTimeout(value, 600)
	}
	const cancel = () => clearTimeout(timer)
	node.addEventListener('pointerdown', start)
	node.addEventListener('pointerup', cancel)
	node.addEventListener('pointerleave', cancel)
})

function App() {
	return (
		<button use:longpress={() => alert('long pressed!')}>
			hold me
		</button>
	)
}

render(<App />)
```
