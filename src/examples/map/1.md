---
title: Reactive map over a list
description: The lower-level primitive behind `<For>` for keyed iteration.
category: reactivity
api: map
import: pota
tags: [map, list, iteration]
---

`map(list, callback)` returns a JSX-shaped reactive accessor that
mirrors `list` (a value or accessor returning the array) and re-uses
DOM rows whose item identity is unchanged — adds, removes, and
reorders are minimal. For most call sites, prefer `<For>` from
`pota/components`, which wraps `map` with prop ergonomics.

```jsx
import { map, render, signal } from 'pota'

function App() {
	const items = signal(['apple', 'banana', 'cherry'])

	return (
		<div>
			<button
				on:click={() =>
					items.update(list => [...list].reverse())
				}
			>
				reverse
			</button>
			<ul>{map(items.read, item => <li>{item}</li>)}</ul>
		</div>
	)
}

render(<App />)
```
