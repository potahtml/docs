title: Custom prop plugin with withValue
description: Build a prop plugin that handles both static values and signals.

`withValue(value, fn)` recursively unwraps functions, arrays, and
promises, then invokes `fn` with the final resolved value. It's the
right helper when authoring a custom prop plugin because it accepts
the same `Attribute<T>` flavor (`T | () => T | Promise<T>`) the rest
of the props pipeline expects.

```jsx
import { propsPlugin, render, signal, withValue } from 'pota'

propsPlugin('use:title', (node, value) => {
	withValue(value, resolved => {
		node.setAttribute('title', String(resolved))
	})
})

function App() {
	const tip = signal('hover me')
	return (
		<div>
			<button use:title={tip.read}>hover</button>
			<button on:click={() => tip.write(`clicked at ${Date.now()}`)}>
				change tooltip
			</button>
		</div>
	)
}

render(<App />)
```
