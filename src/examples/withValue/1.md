---
title: Custom ref factory with withValue
description: Build a use:ref factory that handles both static values and signals.
category: reactivity
api: withValue
import: pota
tags: [withValue, use:ref, factory]
---

`withValue(value, fn)` recursively unwraps functions, arrays, and
promises, then invokes `fn` with the final resolved value. It's
the right helper when authoring a custom `use:ref` factory because
it accepts the same `Attribute<T>` flavor (`T | () => T |
Promise<T>`) the rest of the props pipeline expects.

```jsx
import { render, signal, withValue } from 'pota'

const title = value => node =>
	withValue(value, resolved => {
		node.setAttribute('title', String(resolved))
	})

function App() {
	const tip = signal('hover me')
	return (
		<div>
			<button use:ref={title(tip.read)}>hover</button>
			<button
				on:click={() => tip.write(`clicked at ${Date.now()}`)}
			>
				change tooltip
			</button>
		</div>
	)
}

render(<App />)
```
