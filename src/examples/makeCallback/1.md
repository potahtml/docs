---
title: Normalize children into a callable
description: Wrap children so flow components can re-invoke them with arguments.
category: components
api: makeCallback
import: pota
tags: [makeCallback, children, flow]
---

`makeCallback(children)` returns a function that yields the rendered
children — calling it again re-evaluates them with whatever
arguments you pass. It's the helper flow components like `Show`,
`Switch`, and `For` use to support `children` that are either static
JSX or render functions like `{(item, index) => …}`.

```jsx
import { makeCallback, render } from 'pota'

function HoverPanel(props) {
	const renderChildren = makeCallback(props.children)

	return (
		<div>
			{renderChildren('hovering')}
			{renderChildren('idle')}
		</div>
	)
}

function App() {
	return (
		<HoverPanel>
			{state => <p>state is {state}</p>}
		</HoverPanel>
	)
}

render(<App />)
```
