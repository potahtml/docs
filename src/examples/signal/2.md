---
title: Toggle a boolean signal
description: Flip a boolean with the `update` form.
category: reactivity
api: signal
import: pota
tags: [signal, toggle]
---

`update(prev => next)` is the right call when the new value depends on
the old one — it receives the previous value and is wrapped in
`untrack` internally so reading inside the updater never creates extra
subscriptions.

```jsx
import { render, signal } from 'pota'

function App() {
	const open = signal(false)

	return (
		<div>
			<button on:click={() => open.update(v => !v)}>
				{() => (open.read() ? 'close' : 'open')}
			</button>
			<p>panel is {() => (open.read() ? 'open' : 'closed')}</p>
		</div>
	)
}

render(<App />)
```
