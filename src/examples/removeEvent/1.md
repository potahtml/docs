---
title: Detach an event listener manually
description: Pull a listener off a node before its owner disposes.
category: events
api: removeEvent
import: pota
tags: [removeEvent, listener]
---

`removeEvent(node, type, handler)` removes a listener and returns an
`on` function that re-attaches it. Reach for it when you need to
detach earlier than the owner's cleanup would — e.g. once a one-shot
condition is met.

```jsx
import { addEvent, removeEvent, render, signal } from 'pota'

function App() {
	const seen = signal(0)

	function onMove(e) {
		seen.write(seen.read() === 0 ? e.clientX : seen.read())
		removeEvent(window, 'mousemove', onMove)
	}

	addEvent(window, 'mousemove', onMove)

	return <p>first mouse x: {seen.read}</p>
}

render(<App />)
```
