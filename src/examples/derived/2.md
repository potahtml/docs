---
title: Multi-stage derived chain
description: Run a value through a pipeline of transformations.
category: reactivity
api: derived
import: pota
tags: [derived, chain, pipeline]
---

`derived(f0, f1, f2, ...)` runs the input through each stage in turn.
Each stage receives the previous stage's resolved value. The chain
re-runs whenever any tracked source in any stage changes, but
individual stages can rerun independently when their own deps fire.

```jsx
import { derived, render, signal } from 'pota'

function App() {
	const raw = signal('  Hello, World!  ')

	const cleaned = derived(
		() => raw.read(),
		s => s.trim(),
		s => s.toLowerCase(),
		s => s.replace(/[^a-z0-9]+/g, '-'),
	)

	return (
		<div>
			<input
				prop:value={raw.read}
				on:input={e => raw.write(e.currentTarget.value)}
			/>
			<p>slug: {cleaned}</p>
		</div>
	)
}

render(<App />)
```
