---
title: Async derived with fetch
description: Resolve a fetch chain into a reactive value.
category: reactivity
api: derived
import: pota
tags: [derived, async, fetch]
---

`derived` unwraps promises automatically — each stage's input is
already-resolved by the time it runs. `await d` is also valid: a
derived is thenable and resolves once its current pending stage
commits. `isResolved(d)` is `false` until the chain has committed at
least once.

```jsx
import { derived, isResolved, render, signal } from 'pota'

function App() {
	const id = signal(1)

	const post = derived(
		() => `https://jsonplaceholder.typicode.com/posts/${id.read()}`,
		url => fetch(url),
		res => res.json(),
	)

	return (
		<div>
			<button on:click={() => id.update(n => n + 1)}>next</button>
			<p>
				{() =>
					isResolved(post) ? post().title : 'loading…'
				}
			</p>
		</div>
	)
}

render(<App />)
```
