---
title: Insert children into an existing scope
description: Append reactive children to a parent without creating a new owner.
category: renderer
api: insert
import: pota
tags: [insert, dom]
---

`insert(children, parent?, options?)` is the lower-level cousin of
`render`: it places `children` under `parent` but does **not** wrap
them in a fresh `root`. Use it when you're already inside a reactive
owner (e.g. inside a component) and want to project content into a
different DOM location.

```jsx
import { cleanup, insert, render, signal } from 'pota'

function App() {
	const message = signal('hello from outside')
	const host = document.querySelector('#external')

	insert(<p>{message.read}</p>, host)
	cleanup(() => (host.textContent = ''))

	return <p>main app</p>
}

render(<App />)
```
