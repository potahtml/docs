---
title: Attach an event listener that auto-detaches
description: Listen on `window` or `document` from inside a component and clean up on disposal.
category: events
api: addEvent
import: pota
tags: [addEvent, listener, cleanup]
---

`addEvent(target, type, handler)` adds the listener and registers a
cleanup that removes it when the surrounding owner disposes — saving
you the manual `cleanup(() => removeEventListener(...))` dance.
Helpful for global listeners (`window`/`document`) that components
add to react to outside interactions.

```jsx
import { addEvent, render, signal } from 'pota'

function App() {
	const key = signal('press anything')

	addEvent(window, 'keydown', e => {
		key.write(`pressed: ${e.key}`)
	})

	return <p>{key.read}</p>
}

render(<App />)
```
