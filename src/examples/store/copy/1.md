---
title: Deep-copy with descriptor fidelity
description: Clone a value preserving prototype, accessor snapshots, and frozen state.
category: store
api: copy
import: pota/store
tags: [copy, clone]
---

`copy(value)` is a structural clone with extras: it preserves the
prototype chain (so class instances stay `instanceof`), copies
non-enumerable / symbol keys, snapshots accessor descriptors (the
getter is invoked once inside `untrack`), handles cycles, and
re-applies frozen / sealed / non-extensible state. Built-ins like
`Date`, `RegExp`, and DOM elements pass through by reference.

```js
import { copy } from 'pota/store'

class Point {
	constructor(x, y) {
		this.x = x
		this.y = y
	}
	get magnitude() {
		return Math.hypot(this.x, this.y)
	}
}

const a = new Point(3, 4)
const b = copy(a)

console.log(b instanceof Point) // true
console.log(b.x, b.y, b.magnitude) // 3 4 5
console.log(a === b) // false

const cyclic = { name: 'self' }
cyclic.me = cyclic
const c = copy(cyclic)
console.log(c.me === c) // true — cycle preserved
```
