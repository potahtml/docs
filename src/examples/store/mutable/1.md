---
title: Build a deeply reactive store
description: Wrap an object in a recursive proxy that tracks every read and write.
category: store
api: mutable
import: pota/store
tags: [mutable, store, proxy]
---

`mutable(source, clone?)` returns a deeply-reactive proxy. Reading
any property subscribes the surrounding scope; writing notifies
observers. Nested objects, arrays, `Map`s and `Set`s are reactivated
on access. Pass `clone: true` to deep-copy the input first so
external references can't mutate behind the proxy's back.

```jsx
import { mutable } from 'pota/store'
import { For } from 'pota/components'
import { render } from 'pota'

const todos = mutable([
	{ id: 1, text: 'buy milk', done: false },
	{ id: 2, text: 'walk dog', done: true },
])

function App() {
	return (
		<div>
			<button
				on:click={() =>
					todos.push({
						id: Date.now(),
						text: 'new task',
						done: false,
					})
				}
			>
				add
			</button>
			<ul>
				<For each={() => todos}>
					{todo => (
						<li>
							<input
								type="checkbox"
								prop:checked={() => todo.done}
								on:change={e =>
									(todo.done = e.currentTarget.checked)
								}
							/>
							{() => todo.text}
						</li>
					)}
				</For>
			</ul>
		</div>
	)
}

render(<App />)
```
