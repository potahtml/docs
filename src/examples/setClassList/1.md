---
title: Apply many classes from an object
description: Toggle multiple classes at once using a `{ name: bool }` map.
category: props
api: setClassList
import: pota
tags: [setClassList, css, dom]
---

`setClassList(node, value)` accepts a string, an array of names, or
an object `{ name: truthy }`. Each entry can be a reactive accessor
— signals you reference are tracked and the corresponding class
follows.

```jsx
import { ref, render, setClassList, signal } from 'pota'

function App() {
	const box = ref()
	const open = signal(false)
	const error = signal(false)

	return (
		<div>
			<div use:ref={box} class="panel">
				panel
			</div>
			<button on:click={() => open.update(o => !o)}>
				toggle open
			</button>
			<button on:click={() => error.update(e => !e)}>
				toggle error
			</button>
			{() =>
				setClassList(box(), {
					open: open.read(),
					error: error.read(),
				})
			}
		</div>
	)
}

render(<App />)
```
