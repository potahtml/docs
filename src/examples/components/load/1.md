---
title: Lazily load a route component
description: Code-split with dynamic `import()` inside a `<Route>`.

`load(() => import('./Page.js'))` returns a marked component that
fetches the module the first time it renders. It retries on network
errors (up to nine times) and scrolls to the URL hash once the
loaded module commits. Most useful inside `<Route>` so heavy pages
aren't paid for until visited.

```jsx
import { render } from 'pota'
import { A, Route, load } from 'pota/components'

const Settings = load(() => import('./pages/Settings.js'))
const Reports = load(() => import('./pages/Reports.js'))

function App() {
	return (
		<div>
			<nav>
				<A href="/settings">settings</A>
				{' · '}
				<A href="/reports">reports</A>
			</nav>

			<Route path="/settings">
				<Settings />
			</Route>
			<Route path="/reports">
				<Reports />
			</Route>
		</div>
	)
}

render(<App />)
```
