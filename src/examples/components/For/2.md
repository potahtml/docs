---
title: For with reactive index and fallback
description: Reactively expose row position and show a fallback for empty lists.
category: components
api: For
import: pota/components
tags: [For, fallback, index]
---

Pass `reactiveIndex` and the second callback parameter becomes a
signal accessor — useful when a row needs to display or react to its
own current index (which changes when neighbors are added or
removed). `fallback` renders when the list is empty.

```jsx
import { render, signal } from 'pota'
import { For } from 'pota/components'

function App() {
	const items = signal([])

	return (
		<div>
			<button
				on:click={() =>
					items.update(list => [...list, `item ${list.length + 1}`])
				}
			>
				add
			</button>
			<button on:click={() => items.update(list => list.slice(1))}>
				remove first
			</button>
			<ul>
				<For
					each={items.read}
					reactiveIndex
					fallback={<li>list is empty</li>}
				>
					{(item, index) => (
						<li>
							#{index} — {item}
						</li>
					)}
				</For>
			</ul>
		</div>
	)
}

render(<App />)
```
