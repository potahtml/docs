---
title: Render a keyed list with For
description: Iterate a reactive array and reuse rows by item identity.
category: components
api: For
import: pota/components
tags: [For, list, keyed]
---

`<For each={...}>` renders one child per item, keying by item
identity. When the array is reordered, rows move; when items remain
the same reference, the corresponding DOM node is reused. Pass the
**reader** (`items.read`) to keep the list reactive.

```jsx
import { render, signal } from 'pota'
import { For } from 'pota/components'

function App() {
	const items = signal(['apple', 'banana', 'cherry'])

	return (
		<div>
			<button
				on:click={() =>
					items.update(list => [...list].reverse())
				}
			>
				reverse
			</button>
			<ul>
				<For each={items.read}>
					{item => <li>{item}</li>}
				</For>
			</ul>
		</div>
	)
}

render(<App />)
```
