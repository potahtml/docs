---
title: Conditional rendering with Show
description: Toggle a block on or off based on a signal.
category: components
api: Show
import: pota/components
tags: [Show, conditional]
---

`Show` mounts its children when `when` is truthy and `fallback`
otherwise. Pass the **reader** (`visible.read`) so the condition
stays reactive — `visible` itself is the tuple, and `visible.read()`
would be a one-time snapshot.

```jsx
import { render, signal } from 'pota'
import { Show } from 'pota/components'

function App() {
	const visible = signal(true)

	return (
		<div>
			<button on:click={() => visible.update(v => !v)}>
				toggle
			</button>
			<Show when={visible.read} fallback={<p>hidden</p>}>
				<p>hello</p>
			</Show>
		</div>
	)
}

render(<App />)
```
