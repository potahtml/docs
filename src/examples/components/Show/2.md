---
title: Show with destructured value
description: Use the truthy value via the children render-prop form.
category: components
api: Show
import: pota/components
tags: [Show, render-prop]
---

When `when` is truthy, `Show` invokes the children function with a
reactive accessor for the value — useful when downstream code needs
the value (and TypeScript knows it's non-null inside the branch).

```jsx
import { render, signal } from 'pota'
import { Show } from 'pota/components'

function App() {
	const user = signal(null)

	return (
		<div>
			<button on:click={() => user.write({ name: 'Ada' })}>
				log in
			</button>
			<button on:click={() => user.write(null)}>
				log out
			</button>
			<Show when={user.read} fallback={<p>not logged in</p>}>
				{u => <p>welcome, {() => u().name}</p>}
			</Show>
		</div>
	)
}

render(<App />)
```
