---
title: Coalesce children into a single text
description: Reduce many child expressions to one effect that produces a string.
category: components
api: Normalize
import: pota/components
tags: [Normalize, text]
---

`<Normalize>` resolves all its children and joins them into a single
string-coerced text node. It's the cheap path when you have a
mixture of static text and reactive expressions and don't need
separate text nodes — one effect updates everything together.

```jsx
import { render, signal } from 'pota'
import { Normalize } from 'pota/components'

function App() {
	const first = signal('Ada')
	const last = signal('Lovelace')

	return (
		<div>
			<input
				prop:value={first.read}
				on:input={e => first.write(e.currentTarget.value)}
			/>
			<input
				prop:value={last.read}
				on:input={e => last.write(e.currentTarget.value)}
			/>
			<p>
				<Normalize>
					hello, {first.read} {last.read}!
				</Normalize>
			</p>
		</div>
	)
}

render(<App />)
```
