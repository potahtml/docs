---
title: Pick a component at render time
description: Render a component chosen by a signal — tag string or function alike.
category: components
api: Dynamic
import: pota/components
tags: [Dynamic, polymorphic]
---

`<Dynamic component={value} ...>` renders `value` as a component,
forwarding the rest of its props. `value` can be a tag string
(`'div'`), a function component, a class component, or even an
existing element. Any other prop you pass to `<Dynamic>` is
forwarded verbatim.

```jsx
import { render, signal } from 'pota'
import { Dynamic } from 'pota/components'

function Heading(props) {
	return <h1 style={{ color: 'rebeccapurple' }}>{props.children}</h1>
}

function App() {
	const variant = signal('p')

	return (
		<div>
			<button on:click={() => variant.write('p')}>p</button>
			<button on:click={() => variant.write('strong')}>strong</button>
			<button on:click={() => variant.write(Heading)}>component</button>

			<Dynamic component={variant.read}>
				hello, world
			</Dynamic>
		</div>
	)
}

render(<App />)
```
