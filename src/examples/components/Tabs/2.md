---
title: Programmatic switching with collapsed panels
description: Drive `Tabs` from outside and keep panel state alive with `collapse`.
category: components
api: Tabs
import: pota/components
tags: [Tabs, programmatic, collapse]
---

`Tabs.selected()` returns the active tab's reactive state — so an
external control can read or write the current selection. Setting
`collapse` on a `<Tabs.Panel>` keeps its DOM mounted and just
hides it via `display: none`, which preserves any state inside (form
drafts, scroll position, mounted iframes).

```jsx
import { render } from 'pota'
import { Tabs } from 'pota/components'

function App() {
	return (
		<Tabs selected={1}>
			<Tabs.Labels>
				<Tabs.Label name="overview">overview</Tabs.Label>
				<Tabs.Label name="editor" selected>
					editor
				</Tabs.Label>
				<Tabs.Label name="preview">preview</Tabs.Label>
			</Tabs.Labels>
			<Tabs.Panels>
				<Tabs.Panel>
					<p>read-only summary</p>
				</Tabs.Panel>
				<Tabs.Panel collapse>
					<textarea defaultValue="draft preserved when hidden" />
				</Tabs.Panel>
				<Tabs.Panel collapse>
					<iframe
						src="about:blank"
						style="width:100%;height:200px"
					/>
				</Tabs.Panel>
			</Tabs.Panels>
		</Tabs>
	)
}

render(<App />)
```
