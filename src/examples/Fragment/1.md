---
title: Group elements without a wrapper
description: Return multiple top-level elements from a component.
category: renderer
api: Fragment
import: pota
tags: [Fragment, jsx]
---

`<>...</>` and `<Fragment>...</Fragment>` both group siblings
without inserting an extra DOM node. Use the explicit form when a
tool (eg. an older lint rule) doesn't recognize the shorthand, or
when you want to pass `Fragment` programmatically.

```jsx
import { Fragment, render } from 'pota'

function Header() {
	return (
		<Fragment>
			<h1>title</h1>
			<p>subtitle</p>
		</Fragment>
	)
}

function App() {
	return (
		<>
			<Header />
			<main>body content</main>
		</>
	)
}

render(<App />)
```
