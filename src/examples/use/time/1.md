title: Format times and use auto-disposing timeouts
description: Use the timestamp formatters and `useTimeout`.

`pota/use/time` exposes `now`, `date`, `datetime`, `time`,
`timeWithSeconds`, `day`, plus `useTimeout(cb, delay, ...args)` —
the timeout integrates with the owner so it cancels automatically
on disposal (and `delay` may be reactive). `measure(name, cb)` and
`timing(fn)` are the perf helpers.

```jsx
import { datetime, useTimeout } from 'pota/use/time'
import { render, signal } from 'pota'

function App() {
	const status = signal('click to start')

	function start() {
		status.write(`started at ${datetime()}`)
		useTimeout(() => status.write('done!'), 2000).start()
	}

	return (
		<div>
			<button on:click={start}>start 2s timer</button>
			<p>{status.read}</p>
		</div>
	)
}

render(<App />)
```
