---
title: Snapshot a form into an object
description: Convert a `<form>` element into a plain object with `form2object`.
category: use
api: form
import: pota/use/form
tags: [form, FormData]
---

`form2object(form)` walks the form's `FormData`, collecting fields
into an object — repeated names become arrays automatically.
`object2form(form, obj)` is the inverse, applying values back. Plus
`isDisabled(node)` for a quick `:disabled`-aware check.

```jsx
import { form2object } from 'pota/use/form'
import { render, signal } from 'pota'

function App() {
	const submitted = signal(null)

	return (
		<form
			on:submit={e => {
				e.preventDefault()
				submitted.write(form2object(e.currentTarget))
			}}
		>
			<input name="email" placeholder="email" />
			<label>
				<input type="checkbox" name="topics" value="news" /> news
			</label>
			<label>
				<input type="checkbox" name="topics" value="updates" /> updates
			</label>
			<button>submit</button>
			<pre>{() => JSON.stringify(submitted.read(), null, 2)}</pre>
		</form>
	)
}

render(<App />)
```
