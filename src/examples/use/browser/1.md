---
title: Branch on browser type
description: Use the `isMobile` / `isFirefox` constants to tweak UI.
category: use
api: browser
import: pota/use/browser

---

`isMobile` and `isFirefox` are evaluated once at module load by
sniffing `navigator.userAgent`. They're plain booleans — handy for
small UA-conditional tweaks (mobile-specific copy, Firefox quirk
fallbacks).

```jsx
import { isFirefox, isMobile } from 'pota/use/browser'
import { render } from 'pota'

function App() {
	return (
		<div>
			<p>device: {isMobile ? 'mobile' : 'desktop'}</p>
			{isFirefox ? (
				<p>Hi Firefox user!</p>
			) : (
				<p>Hi non-Firefox user!</p>
			)}
		</div>
	)
}

render(<App />)
```
