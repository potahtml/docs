---
title: Per-row selection state with useSelector
description: Build a one-of-many selection without per-row effects.
category: use
api: selector
import: pota/use/selector
tags: [selector, list, selection]
---

`useSelector(currentSignal)` returns an `isSelected(item)` factory:
each row asks `isSelected(item)` once and gets a stable boolean
signal that flips when the parent's `currentSignal` changes — only
the previously-selected and newly-selected rows re-run, not all
rows.

```jsx
import { useSelector } from 'pota/use/selector'
import { For } from 'pota/components'
import { render, signal } from 'pota'

const items = ['apple', 'banana', 'cherry', 'date']

function App() {
	const current = signal('apple')
	const isSelected = useSelector(current.read)

	return (
		<ul>
			<For each={items}>
				{item => (
					<li
						class={() => (isSelected(item)() ? 'selected' : '')}
						on:click={() => current.write(item)}
					>
						{item}
					</li>
				)}
			</For>
		</ul>
	)
}

render(<App />)
```
