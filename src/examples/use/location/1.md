---
title: Read and navigate the URL reactively
description: Use the reactive `location` object plus `navigate()` for programmatic routing.
category: use
api: location
import: pota/use/location
tags: [location, routing, url]
---

`location` is a frozen object of accessors over `window.location` —
read `location.pathname()`, `location.hash()`, `location.search()`,
or the reactive `location.searchParams` map. `navigate(path,
options)` performs client-side navigation; `addListeners()` ensures
`popstate`/click handlers are wired up.

```jsx
import {
	addListeners,
	location,
	navigate,
} from 'pota/use/location'
import { render } from 'pota'

addListeners()

function App() {
	return (
		<div>
			<p>path: {location.pathname}</p>
			<p>hash: {location.hash}</p>
			<button on:click={() => navigate('/about')}>
				go to /about
			</button>
			<button on:click={() => navigate('/users/42')}>
				go to /users/42
			</button>
		</div>
	)
}

render(<App />)
```
