---
title: Two-way input binding with use:bind
description: Mirror an input's value into a `bind`-derived signal.
category: use
api: bind
import: pota/use/bind
tags: [bind, form, input]
---

Importing `pota/use/bind` registers the `use:bind` plugin. `bind(initial)`
returns a writable derived — read with `value()`, write with `value(v)`
— that the plugin keeps in sync with whatever input it's attached to
(text, checkbox, radio, contenteditable). One signal, two-way data flow.

```jsx
import { bind } from 'pota/use/bind'
import { render } from 'pota'

function App() {
	const name = bind('world')
	const subscribed = bind(false)

	return (
		<form>
			<label>
				name: <input use:bind={name} />
			</label>
			<label>
				<input type="checkbox" use:bind={subscribed} /> subscribe
			</label>
			<p>
				hello {() => name()}, subscribed={() => String(subscribed())}
			</p>
		</form>
	)
}

render(<App />)
```
