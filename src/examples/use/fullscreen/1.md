---
title: Toggle fullscreen on click
description: Make any element fullscreen-ready via a `use:` directive.
category: use
api: fullscreen
import: pota/use/fullscreen
tags: [fullscreen]
---

Importing `pota/use/fullscreen` registers `use:fullscreen` — clicking
the element toggles fullscreen. `use:fullscreen={true}` toggles the
element itself; pass a function to choose a different target. The
`useFullscreen()` accessor reports the current fullscreen element
(or `null`).

```jsx
import { ref, render } from 'pota'
import { useFullscreen } from 'pota/use/fullscreen'
import 'pota/use/fullscreen'

function App() {
	const stage = ref()
	const current = useFullscreen()

	return (
		<div>
			<button use:fullscreen={() => stage()}>
				toggle fullscreen
			</button>
			<div
				use:ref={stage}
				style={{
					background: 'rebeccapurple',
					color: 'white',
					padding: '2rem',
				}}
			>
				stage — fullscreen target
			</div>
			<p>currently fullscreen: {() => (current() ? 'yes' : 'no')}</p>
		</div>
	)
}

render(<App />)
```
