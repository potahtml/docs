---
title: Toggle fullscreen on click
description: Make any element fullscreen-ready via a ref factory.
category: use
api: fullscreen
import: pota/use/fullscreen
tags: [fullscreen, use:ref]
---

`pota/use/fullscreen` exports a `fullscreen(target?)` factory that
returns a ref function. Attach it via `use:ref` and clicking the
element toggles fullscreen. With no argument the element itself
goes fullscreen; pass an element or a function returning one to
target something else. The `useFullscreen()` accessor reports the
current fullscreen element (or `null`).

```jsx
import { ref, render } from 'pota'
import { fullscreen, useFullscreen } from 'pota/use/fullscreen'

function App() {
	const stage = ref()
	const current = useFullscreen()

	return (
		<div>
			<button use:ref={fullscreen(() => stage())}>
				toggle fullscreen
			</button>
			<div
				use:ref={stage}
				style={{
					background: 'rebeccapurple',
					color: 'white',
					padding: '2rem',
				}}
			>
				stage — fullscreen target
			</div>
			<p>currently fullscreen: {() => (current() ? 'yes' : 'no')}</p>
		</div>
	)
}

render(<App />)
```
