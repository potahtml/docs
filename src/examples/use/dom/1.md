---
title: Inspect what's connected and focused
description: Use the DOM helpers `isConnected` and `activeElement`.
category: use

import: pota/use/dom

---

`pota/use/dom` re-exports a small set of cached / safely-bound DOM
helpers used internally — `document`, `head`, `documentElement`,
`createElement`, `isConnected`, `activeElement`, etc. They're
exposed because the bound forms are slightly faster on hot paths and
guard against missing globals (e.g. SSR-ish smoke tests).

```jsx
import { activeElement, isConnected } from 'pota/use/dom'
import { ready, ref, render } from 'pota'

function App() {
	const el = ref()

	ready(() => {
		console.log('connected?', isConnected(el()))
		el().focus()
		console.log('active?', activeElement() === el())
	})

	return <input use:ref={el} placeholder="focused on mount" />
}

render(<App />)
```
