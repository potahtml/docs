---
title: Scroll to a target programmatically
description: Use `scrollToSelector`, `scrollToTop`, and friends.
category: use
api: scroll
import: pota/use/scroll
tags: [scroll, navigation]
---

`scrollToElement(node)` resets the node's `scrollTop` and calls
`scrollIntoView`. `scrollToSelector(sel)` finds and scrolls to a
selector (returning whether it succeeded), and
`scrollToSelectorWithFallback` falls back to the top if the selector
isn't found. `scrollToTop` scrolls the window to the top.

```jsx
import {
	scrollToSelectorWithFallback,
	scrollToTop,
} from 'pota/use/scroll'
import { render } from 'pota'

function App() {
	return (
		<div>
			<nav>
				<button
					on:click={() =>
						scrollToSelectorWithFallback('#section-2')
					}
				>
					jump to section 2
				</button>
				<button on:click={scrollToTop}>top</button>
			</nav>
			<section style={{ height: '100vh' }}>section 1</section>
			<section id="section-2" style={{ height: '100vh' }}>
				section 2
			</section>
		</div>
	)
}

render(<App />)
```
