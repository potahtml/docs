---
title: Share a subscription across components with Emitter
description: Hook up to `window`/`document`/etc once and broadcast via a signal.
category: use
api: emitter
import: pota/use/emitter
tags: [Emitter, subscription, shared]
---

`new Emitter({ on, initialValue })` builds a sharable hook: `on` is
called once when the first consumer subscribes (it returns a
disposer), `initialValue` seeds the signal. `emitter.use()` returns
a reader you can drop into JSX; `emitter.on(fn)` runs `fn` on every
change. Reference-counted — when the last consumer disposes, the
underlying subscription is torn down.

```jsx
import { Emitter } from 'pota/use/emitter'
import { render } from 'pota'

const onlineEmitter = new Emitter({
	on: dispatch => {
		const update = () => dispatch(navigator.onLine)
		window.addEventListener('online', update)
		window.addEventListener('offline', update)
		return () => {
			window.removeEventListener('online', update)
			window.removeEventListener('offline', update)
		}
	},
	initialValue: () => navigator.onLine,
})

function App() {
	const online = onlineEmitter.use()
	return <p>connection: {() => (online() ? 'online' : 'offline')}</p>
}

render(<App />)
```
