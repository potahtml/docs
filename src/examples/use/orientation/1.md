---
title: Reactive orientation accessor
description: Branch layout on landscape vs portrait viewport.
category: use
api: orientation
import: pota/use/orientation
tags: [orientation, layout]
---

`useOrientation()` returns a signal reader that's `'horizontal'`
when the viewport is at least as wide as it is tall, otherwise
`'vertical'`. It's reference-counted under the hood — only one
underlying `resize` listener exists no matter how many components
subscribe.

```jsx
import { useOrientation } from 'pota/use/orientation'
import { Match, Switch } from 'pota/components'
import { render } from 'pota'

function App() {
	const orientation = useOrientation()

	return (
		<Switch>
			<Match when={() => orientation() === 'horizontal'}>
				<p>side-by-side layout (landscape)</p>
			</Match>
			<Match when={() => orientation() === 'vertical'}>
				<p>stacked layout (portrait)</p>
			</Match>
		</Switch>
	)
}

render(<App />)
```
