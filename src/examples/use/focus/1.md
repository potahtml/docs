---
title: Move focus next/previous and observe document focus
description: Provide custom focus traversal and react to window focus changes.
category: use
api: focus
import: pota/use/focus
tags: [focus, accessibility]
---

`focusNext()` / `focusPrevious()` cycle focus through tabbable
elements (skipping disabled). `useDocumentFocus()` is a signal-style
accessor that flips between `true` (window focused) and `false`.

```jsx
import {
	focusNext,
	focusPrevious,
	useDocumentFocus,
} from 'pota/use/focus'
import { render } from 'pota'

function App() {
	const isFocused = useDocumentFocus()

	return (
		<div>
			<p>window is {() => (isFocused() ? 'focused' : 'blurred')}</p>
			<input placeholder="one" />
			<input placeholder="two" />
			<input placeholder="three" />
			<button on:click={() => focusPrevious()}>← prev</button>
			<button on:click={() => focusNext()}>next →</button>
		</div>
	)
}

render(<App />)
```
