---
title: Manage MediaStream tracks
description: Copy or stop audio/video tracks across `MediaStream` instances.
category: use
api: stream
import: pota/use/stream
tags: [stream, media, webrtc]
---

`pota/use/stream` provides a tiny set of `MediaStream` helpers:
`copyAudioTracks` / `copyVideoTracks` clone tracks from one stream
to another, the matching `removeAudioTracks` / `removeVideoTracks`
detach them, and `stopStream(stream)` (or `stopTracks(stream)` /
`stopTrack(track)`) end every track on a stream so the camera/mic
indicator turns off.

```js
import { copyVideoTracks, stopStream } from 'pota/use/stream'

const camera = await navigator.mediaDevices.getUserMedia({
	video: true,
	audio: true,
})

const videoOnly = new MediaStream()
copyVideoTracks(camera, videoOnly)

document.querySelector('video').srcObject = videoOnly

// later, when done:
setTimeout(() => stopStream(camera), 5000)
```
