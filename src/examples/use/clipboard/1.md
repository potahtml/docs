---
title: Copy to clipboard on click
description: Copy text to the clipboard via a single declarative prop.
category: use
api: clipboard
import: pota/use/clipboard
tags: [clipboard, copy]
---

Importing `pota/use/clipboard` registers a `use:clipboard` plugin.
Pass `true` to copy the element's `innerText`, a string to copy that
literal text, or a function `(event) => string` to compute it on the
fly.

```jsx
import { render } from 'pota'
import 'pota/use/clipboard'

function App() {
	return (
		<div>
			<button use:clipboard={true}>
				copy this label's text
			</button>
			<button use:clipboard="hard-coded snippet">
				copy a fixed string
			</button>
			<button use:clipboard={() => `time: ${Date.now()}`}>
				copy current time
			</button>
		</div>
	)
}

render(<App />)
```
