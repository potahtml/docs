---
title: Copy to clipboard on click
description: Copy text to the clipboard via a single ref factory.
category: use
api: clipboard
import: pota/use/clipboard
tags: [clipboard, copy, use:ref]
---

`pota/use/clipboard` exports a `clipboard(value)` factory that
returns a ref function. Attach it via `use:ref`. Pass `true` to
copy the element's `innerText`, a string/number to copy that
literal value, or a function `(event) => string | number` to
compute it on the fly.

```jsx
import { render } from 'pota'
import { clipboard } from 'pota/use/clipboard'

function App() {
	return (
		<div>
			<button use:ref={clipboard(true)}>
				copy this label's text
			</button>
			<button use:ref={clipboard('hard-coded snippet')}>
				copy a fixed string
			</button>
			<button use:ref={clipboard(() => `time: ${Date.now()}`)}>
				copy current time
			</button>
		</div>
	)
}

render(<App />)
```
