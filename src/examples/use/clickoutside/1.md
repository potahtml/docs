---
title: Close a popover on outside click
description: Dismiss a UI element when the user clicks anywhere outside it.
category: use
api: clickoutside
import: pota/use/clickoutside
tags: [clickoutside, popover]
---

Importing `pota/use/clickoutside` registers two plugins:
`use:clickoutside` for repeated firing and `use:clickoutsideonce` for
single-shot dismissal. The handler runs whenever a `pointerdown`
lands outside the element it's attached to.

```jsx
import { render, signal } from 'pota'
import { Show } from 'pota/components'
import 'pota/use/clickoutside'

function App() {
	const open = signal(false)

	return (
		<div>
			<button on:click={() => open.write(true)}>open</button>
			<Show when={open.read}>
				<div use:clickoutside={() => open.write(false)}>
					<p>I close on outside click.</p>
				</div>
			</Show>
		</div>
	)
}

render(<App />)
```
