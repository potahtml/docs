---
title: Author a stylesheet with the css template literal
description: Build a `CSSStyleSheet` you can adopt into a document or shadow root.
category: use
api: css
import: pota/use/css
tags: [css, stylesheet]
---

`css` is a tagged template that returns a cached `CSSStyleSheet`.
Pair it with `addAdoptedStyleSheet(target, sheet)` to mount the
sheet on the document or a shadow root — the result is one shared
sheet across all consumers, no `<style>` tag duplication.

```js
import { addAdoptedStyleSheet, css } from 'pota/use/css'

const buttons = css`
	button {
		font-weight: 600;
		border-radius: 6px;
	}
	button.primary {
		background: rebeccapurple;
		color: white;
	}
`

addAdoptedStyleSheet(document, buttons)

document.body.innerHTML = `
	<button>plain</button>
	<button class="primary">primary</button>
`
```
