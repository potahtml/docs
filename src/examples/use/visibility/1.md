---
title: Pause work when the tab is hidden
description: React to `document.visibilityState` via a shared accessor.
category: use
api: visibility
import: pota/use/visibility
tags: [visibility, document]
---

`useDocumentVisible()` returns a reader that's `true` when the tab
is visible and `false` when it's backgrounded — so you can pause
animations, polling, or media playback while the user isn't
looking. `isDocumentVisible()` is the one-shot equivalent.

```jsx
import { useDocumentVisible } from 'pota/use/visibility'
import { cleanup, effect, render, signal } from 'pota'

function App() {
	const visible = useDocumentVisible()
	const ticks = signal(0)

	effect(() => {
		if (!visible()) return
		const id = setInterval(() => ticks.update(n => n + 1), 250)
		cleanup(() => clearInterval(id))
	})

	return (
		<p>
			ticks: {ticks.read} (only counts while tab is visible)
		</p>
	)
}

render(<App />)
```
