---
title: Emit a custom event between components
description: Dispatch a typed event upward through the DOM tree.
category: use
api: event
import: pota/use/event

---

`emit(node, name, init?)` dispatches a `CustomEvent` that bubbles,
is cancelable, and is composed by default. Listen for it on any
ancestor with `on:<name>`. This is the idiomatic way to bubble a
"the inner widget did X" signal up to a parent without prop
drilling or callback chains.

```jsx
import { emit } from 'pota/use/event'
import { render, signal } from 'pota'

function StarButton(props) {
	return (
		<button
			on:click={e => emit(e.currentTarget, 'app:starred', { detail: props.id })}
		>
			★ star {props.id}
		</button>
	)
}

function App() {
	const lastStarred = signal('—')

	return (
		<section
			on:app:starred={e => lastStarred.write(String(e.detail))}
		>
			<StarButton id={1} />
			<StarButton id={2} />
			<StarButton id={3} />
			<p>last starred: {lastStarred.read}</p>
		</section>
	)
}

render(<App />)
```
