title: Stop and emit DOM events

`pota/use/event` collects ergonomic versions of the `Event`
operations: `preventDefault`, `stopPropagation`,
`stopImmediatePropagation`, the combined `stopEvent`, and `emit` for
dispatching custom events that bubble + are composed by default.

```jsx
import { emit, stopEvent } from 'pota/use/event'
import { render } from 'pota'

function App() {
	return (
		<form
			on:submit={e => {
				stopEvent(e)
				emit(e.currentTarget, 'app:submit', { detail: 'ok' })
			}}
			on:app:submit={e => alert('intercepted: ' + e.detail)}
		>
			<button>submit</button>
		</form>
	)
}

render(<App />)
```
