---
title: Track the viewport size
description: React to viewport `resize` via a shared signal accessor.
category: use
api: resize
import: pota/use/resize
tags: [resize, viewport]
---

`useDocumentSize()` returns a reader for `{ width, height }` of the
document element — updated on `window.resize`. Like the other
`Emitter`-backed hooks, the underlying listener is reference-counted
across consumers.

```jsx
import { useDocumentSize } from 'pota/use/resize'
import { render } from 'pota'

function App() {
	const size = useDocumentSize()
	return (
		<p>
			viewport is {() => size().width}×{() => size().height}
		</p>
	)
}

render(<App />)
```
