`pota/use/random` exposes a small kit of crypto-grade helpers:
`random()` for `[0, 1)`, `randomBetween(min, max)` for inclusive
integers, `randomColor(min, max)` for an RGB string, `randomId()` for
a base36 id, `chance(percent)` for a weighted boolean, and
`randomSeeded(seed)` for a deterministic generator.

```jsx
import {
	chance,
	randomBetween,
	randomColor,
	randomId,
} from 'pota/use/random'
import { render, signal } from 'pota'

function App() {
	const id = signal(randomId())
	const color = signal(randomColor())

	function shuffle() {
		id.write(randomId())
		color.write(randomColor())
	}

	return (
		<div>
			<p>id: <code>{id.read}</code></p>
			<div style={{ background: color.read, padding: '0.5rem' }}>
				color: {color.read}
			</div>
			<p>roll d6: {() => randomBetween(1, 6)}</p>
			<p>50/50: {() => (chance(50) ? 'yes' : 'no')}</p>
			<button on:click={shuffle}>shuffle</button>
		</div>
	)
}

render(<App />)
```
