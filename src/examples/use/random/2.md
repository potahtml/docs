---
title: Seeded RNG for deterministic output
description: Use `randomSeeded(seed)` so tests and replays produce the same sequence.
category: use
api: random
import: pota/use/random
tags: [random, seeded, determinism]
---

`randomSeeded(seed)` returns a function with the same shape as
`random()` but seeded — same seed, same sequence. Useful for tests,
shareable demos ("everyone with this seed sees the same dungeon"),
and snapshot reproducibility.

```jsx
import { randomBetween, randomSeeded } from 'pota/use/random'
import { For } from 'pota/components'
import { render, signal } from 'pota'

function App() {
	const seed = signal(42)

	const numbers = () => {
		const rnd = randomSeeded(seed.read())
		return Array.from(
			{ length: 8 },
			() => randomBetween(1, 100, rnd),
		)
	}

	return (
		<div>
			<label>
				seed:{' '}
				<input
					type="number"
					prop:value={seed.read}
					on:input={e =>
						seed.write(Number(e.currentTarget.value))
					}
				/>
			</label>
			<ul>
				<For each={numbers}>{n => <li>{n}</li>}</For>
			</ul>
		</div>
	)
}

render(<App />)
```
