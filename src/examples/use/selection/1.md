---
title: Select all of an element on click
description: Use the `clickSelectsAll` ref and the selection helpers.
category: use
api: selection
import: pota/use/selection
tags: [selection, range, use:ref]
---

`pota/use/selection` exports `clickSelectsAll` — a ref function
that selects all of an element's text on click. The module also
exports `getSelection()` / `restoreSelection(range)` for capturing
and re-applying a `Range` (used internally by `bind` to keep
contenteditable cursors stable).

```jsx
import { render } from 'pota'
import { clickSelectsAll } from 'pota/use/selection'

function App() {
	return (
		<div>
			<p>click the snippet to select it all:</p>
			<code use:ref={clickSelectsAll}>npm install pota</code>
		</div>
	)
}

render(<App />)
```
