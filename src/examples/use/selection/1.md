---
title: Select all of an element on click
description: Use the `use:click-selects-all` plugin and the selection helpers.
category: use
api: selection
import: pota/use/selection
tags: [selection, range]
---

Importing `pota/use/selection` registers `use:click-selects-all`,
which selects all of an element's text on click. The module also
exports `getSelection()` / `restoreSelection(range)` for capturing
and re-applying a `Range` (used internally by `bind` to keep
contenteditable cursors stable).

```jsx
import { render } from 'pota'
import 'pota/use/selection'

function App() {
	return (
		<div>
			<p>
				click the snippet to select it all:
			</p>
			<code use:click-selects-all>
				npm install pota
			</code>
		</div>
	)
}

render(<App />)
```
