`pota/use/string` collects small string utilities:
`dashesToCamelCase`, `capitalizeFirstLetter`, `label` (kebab/snake →
spaces), `short` (truncate with ellipsis), `ensureString`,
`toString` (trim+optional-cap), `copyToClipboard`, and an async
`hash(value, algo?)` that returns a hex digest.

```jsx
import {
	capitalizeFirstLetter,
	hash,
	label,
	short,
} from 'pota/use/string'
import { render, signal } from 'pota'

function App() {
	const sha = signal('')

	hash('hello world').then(sha.write)

	return (
		<div>
			<p>{capitalizeFirstLetter('hello, world')}</p>
			<p>label: {label('hello-world_demo')}</p>
			<p>
				short:{' '}
				{short(
					'this is a long sentence that the helper will truncate at the 40-character mark',
				)}
			</p>
			<p>sha-256(hello world): {() => short(sha.read())}</p>
		</div>
	)
}

render(<App />)
```
